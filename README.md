### 一、相关技术

- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`

### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示

系统分为：用户、商家、管理员。

用户功能：1. 登录、2. 注册、3. 社区商业中心、4. 我的购物车、5. 我的订单、6. 历史订单

商家功能：1. 登录、2. 商品管理、3. 订单管理、4. 店铺信息、5. 销售汇总

管理员功能：1. 登录、2. 顾客管理、3. 商家管理、4. 店铺管理
